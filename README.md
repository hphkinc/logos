# Logos

## Usage

- Should be lowercase:

  ```
  https://logos.hphk.io/{type}/{name}.{jpg|png}
  ```

- If you want to download:

  ```bash
  curl https://logos.hphk.io/s/python.jpg --output output.jpg
  ```
